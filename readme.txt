== Description ==

IzzyHelp is online service to support your visitors, more information on [IzzyHelp.com](http://www.IzzyHelp.com), it's available as 30-days free trial, then hosted in IzzyHelp's cloud or on your own PHP server.

== Installation ==

Upload the IzzyHelp plugin to your blog, Activate it, then enter your [IzzyHelp.com user ID and helpdesk ID](http://www.IzzyHelp.com/), you'll find them on IzzyHelp dashboard, under Installation tab.

1, 2, 3: You're done!
